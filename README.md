## 프로젝트 시작 (로컬)

1. 프로젝트를 clone 합니다.

```jsx
git clone https://github.com/Happy-New-Here/HappyNewHere_FE.git
```

2. clone이 완료되었다면, 다음 명령어로 npm library들을 설치합니다.

```jsx
npm i
```

1. 프로젝트를 실행합니다.

```jsx
npm run dev
```
